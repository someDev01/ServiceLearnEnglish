# ServiceLearnEnglish

ServiceLearnEnglish - это сервис для изучения английского языка через просмотр коротких видео с удобными субтитрами, удобным словарем, где можно создавать записи слов для запоминания, а также интеграция с внешними сервисами перевода и транскрипции

ПРОЕКТ ЕЩЕ НАХОДИТСЯ В СТАДИИ РАЗРАБОТКИ!!! Это не конечная версия продукта. Все что сейчас есть в проекте, может быть изменено или переделано.

## Основной функционал
- Регистрация и авторизация по email-коду без пароля
- JWT-аутентификация
- Просмотр контента
- Пагинация для оптимизированного способа получения контента на сервисе
- Избранное контента
- Интерактивное взаимодействие с субтитрами
- Создание словаря с заметками
- Удобные заметки с удобным запоминанием, а также с источником отуда вся то слово( из какого видео-контента)
- Интеграция с внешними API перевода
- Интеграция с API транскрипции
- Кэширование данных которые получает пользователь
- Глобальная обработка ошибок

## Бизнес-логика
-Пользователь регистрируется по passwordless(без пароля) системе с email-кодом.
-Вход для пользователя реализован через интерйес сервиса.
-Вход для администратора реализован только через инструменты(swagger, postman и подобные), так как помимо email-кода для администратора требуется ввести секретный ключ
-Просмотр видео доступен только авторизованным пользователям.
-Просмотр подробностей контента доступно без авторизации
-Добавление заметок привязано к пользователю и также доступно только авторизованным пользователям
-Есть поикс для быстрой навигации конетента.
-В словарь можно добавлять заметки с словами для изучения
-Добавлять контент и изменять его может только администратор
